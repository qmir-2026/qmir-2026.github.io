Getting to know R

Goals: - Overall - Git/GitHub - Quarto - R

Week 1: Welcome, Getting to know each other, Course Goals (Example Quarto Manuscript), Structure Goals: - Overall: Students know course goals, expectation alignment - Git/GitHub: Students know what these terms mean - Quarto: Students know what this term means - R: Students know what R is

Week 2: Reproducible Computational Research Goals: - Overall: Students can set up a repository and create basic Quarto documents within Positron - Git: Students can use git init, git add and git commit (hw github) - Quarto: Students can write simple manuscripts with Quarto and render it to PDF - R: Students know basic R syntax

Week 3: R Fundamentals: How computers store and handle information Goals: - Overall: Students understand how computers store information, know basic data objects and functions in R and - Git - Quarto: Students learn how code cells function, cell options - R: Learn basic variable types (numeric, character strings, logical, dates and times), how they are stored internally, vectors, matrix, dataframe, list, array

Week 4: Tidyverse: Importing, Wrangling, and Visualizing Data Goals: - Overall: Students know what tidy data is, learn basic dplyr functions and ggplot - Quarto: Students can create simple tables and figures in Quarto - R: Students can import, manipulate, summarize and visualize data

Week 5: Tidyverse: Reshape and Merge Data Goals: - Overall: Students learn how to turn non-tidy data formats into tidy (long-)format and combine data from different sources - Git: - Quarto: - R:

Week 6: Probabilities, Distributions, and Bayes Theorem Goals: - Overall: Understand what probabilities and distributions are and that there many different, how to express probabilities and beliefs as distributions, value of simulations, prior, likelihood, posterior - Git: - Quarto: - R: Students can simulate random draws, visualize distributions

Week 7: Learning from Data: Markov Chain Monte Carlo Simulation - Overall: Students understand the (basic) logic of Bayesian data analysis and MCMC algorithms to sample from the posterior - Git: - Quarto: Students learn a bayesian workflow scheme - R:

Week 8: Bivariate Linear Regression - Overall: Students understand the conceptual logic of linear regression, OLS as best line, MCMC for collecting the distribution of plausible lines - Git: Students learn how to use git LFS? - Quarto: - R: Students can implement a linear regression model using brms, assess (non-)convergence, explore results

Week 9: Multivariate Linear Regression - Overall: Students understand the conceptual logic of including multiple variables, conditional effects, interactions with dummy, categories and numerical variables - Git: Students learn how to use git LFS? - Quarto: Students learn how to report model estimates - R: Students can implement a linear regression model using brms, assess (non-)convergence, explore results

Week 10: Modeling Binary Outcomes - Overall: Students understand the conceptual logic of working with binary dependent variables - Git: - Quarto: Students practise the bayesian workflow using binary outcome - R: Students can implement and interpret a binary outcome model

Week 11: Fixed Effects Models - Overall: Studens learn the concept of hierarchical data, Students understand the conceptual logic of fixed effects, need for IR/PS datasets - Git: - Quarto: Students practise the bayesian workflow using fixed effects model - R: Students can implement and interpret a fixed effects (time series?) model

Week 12: Multilevel Models - Overall: Studens reiterate the concept of hierarchical data, Students understand the conceptual logic of partial pooling, contrast with FE () and Complete Pooling - Git: - Quarto: Students practise the bayesian workflow using partial pooling (mixed effects model) - R: Students can implement and interpret a partial pooling model

Week 13: Semester Wrap-Up and Moving On - Overall: Outlook, tips for Take Home Exam - Git: - Quarto: Manuscript as ideal, overkill for you mostly, QuickQuartoSetup -> focus on PDF only - R: AI and Code
